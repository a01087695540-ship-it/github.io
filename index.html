Main · 교육안내

AX 이해를 넘어, 리더의 실행 언어로 전환합니다.

본 과정은 단순한 AI 트렌드 학습이 아니라, 리더가 조직 내 AX 전환을 어떻게 촉진하고 실제 업무 변화로 연결할 것인지에 초점을 둡니다.

01

AX 방향성 설정

AX의 개념과 산업 내 변화 흐름을 이해하고, 롯데GRS 관점에서 적용 가능한 전환 방향을 탐색합니다.

02

리더 역할 정립

일하는 방식의 변화를 이끄는 리더의 역할을 점검하고, 조직 내 실행과 확산을 위한 가이드라인을 확인합니다.

03

데이터 기반 의사결정

AI 도구와 Agent 활용 사례를 통해 데이터 중심의 의사결정 문화를 조직에 정착시키는 방안을 학습합니다.

Program Feature

실행 중심, 프로젝트형 AX 리더십 경험

이론 중심 교육에서 벗어나 AI 시연, 업무 적용 방식, 후속 AX 부스터 과정까지 연결되는 구조로 설계했습니다.

AX를 이해하고,
현업 적용 가능성을 확인합니다.

리더가 먼저 AI 활용 경험을 갖고, 조직 구성원이 실제 업무에서 AI를 활용할 수 있도록 방향성과 실행 기준을 제시합니다.

Leadership

변화 촉진자로서의 리더

리더가 AX 전환의 필요성을 이해하고, 업무 방식 변화의 기준을 제시할 수 있도록 구성했습니다.

Use Case

산업별 AI 활용 사례

산업과 Value Chain 관점의 AX 사례를 통해 현업 적용 가능성을 구체적으로 살펴봅니다.
